# Futuristic-Career-Skill-Builder
⚡ Futuristic Career & Skill Builder

An AI-powered web app that helps you discover the right tech career path, learn from your own notes, and improve through smart quizzes — all in one place.

✨ What It Does
🎯 Find Your Career Path
Answer a few questions and get personalized tech career recommendations with skills and learning paths.
📚 Learn From Your Own Notes
Upload PDFs, DOCX, or TXT files and let AI:
Summarize your content
Answer questions
Extract key insights
🧩 Test Yourself with AI Quizzes
Generate quizzes from your notes and track your mistakes to improve faster.
💬 Interactive Learning Experience
Keeps your progress, answers, and history so you can continue anytime.
🛠 Built With
Streamlit
Python
SQLite
FAISS
Sentence Transformers
Google Gemini API
